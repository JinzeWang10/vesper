# rixing （日省）— skill skeleton（v0，占位）

> ⚠️ 这是骨架占位。完整行为待 `docs/plans/2026-06-17-rixing-v0-design.md` 写出实现计划后填充。

## 触发

- 自动：每天收工时（默认傍晚 session 结束 + 当日未问过），抛出当天的一个问题。
- 手动：用户主动 `/收工`（或等价命令）随时领取当天问题。
- 频率铁律：**每天只问一次**。

## 流程（待实现）

1. 读本地历史回答（`data/`，gitignore）。
2. 基于历史生成一个苏格拉底式、向内的问题（带出处引用旧答案）。
3. 呈现问题 —— 极简、沉浸、放松。
4. 接收回答。
5. **当场不回应**，只温柔收下（如「记住了，明天见」）。
6. 落本地存储。

## 周期回看（待实现）

- 每周/每月触发「镜子时刻」：回看窗口内回答，归纳模式并带出处呈现。

## 不变量

- 每天一次、答完不回应、无打卡/分数/社交、local-first、可跳过不追问。
- 触发时机与口吻可由用户自定义。
